# Escape Routes (From My Own Head)

This directory contains original lyrics for a concept album about unlearning bad habits and taking time away from others to grow. Each track blends humor, honesty, and introspection, channeling influences from artists like Julia Michaels and Juice WRLD without mentioning them directly.

The main tracklist lives in `tracklist.md`. There are also two alternate takes on the album:

- `hiphop_grunge/`: hip hop infused emo-grunge versions for each song.
- `country_emo/`: twangy country-emo renditions of the same tracks.

Use these files for inspiration or adaptation to fit your own style.
